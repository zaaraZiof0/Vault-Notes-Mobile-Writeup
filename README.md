A recovered Android application package was partially decoded after extraction. The application files are incomplete but still contain enough logic and data artifacts to reconstruct the hidden token used by the app. Analyze the provided Android app artifacts and recover the correct flag.

# CTF Writeup: Vault Notes

**Category:** Android Mobile  
**Difficulty:** Medium  
**Flag:** `ZeroX{m0************************_x7}`

---

## 1. Challenge Overview
The **Vault Notes** challenge requires recovering a flag split into three 8-byte segments hidden across different Android application artifacts. The goal is to identify these fragments and reverse their specific obfuscation methods.

## 2. Environment & Tools
- **OS:** Kali Linux
- **Tools:** `apktool` (for decoding), `nano`, `python3`

---

## 3. Step-by-Step Solution

### Segment 1: Asset Analysis (`assets/session.dat`)
The first segment is located in the `assets` folder. The file `session.dat` contains a string that is Base64 encoded and then XORed with `0x17`.

**Transformation:**
1. Base64 decode the content of `session.dat`.
2. XOR each resulting byte with `0x17`.

**Result:** `m0******`

---

### Segment 2: Resource Analysis (`res/values/strings.xml`)
The second segment is stored as a numeric vector in the strings resource file under the name `cache_vector`.

**Data Found:**
`117,119,52,105,55,102,119,98`

**Transformation:**
Subtract 3 from each decimal value and convert the result to ASCII characters:
- 117 - 3 = 114 (**r**)
- 119 - 3 = 116 (**t**)
- 52 - 3 = 49 (**1**)
- 105 - 3 = 102 (**f**)
- 55 - 3 = 52 (**4**)
- 102 - 3 = 99 (**c**)
- 119 - 3 = 116 (**t**)
- 98 - 3 = 95 (**_**)

**Result:** `rt********`

---

### Segment 3: Bytecode Analysis (`smali/com/zerox/vault/GateKeeper.smali`)
The final segment is generated by logic found in the `GateKeeper` class. It uses a hex "mask" and a "key" string.

**Variables:**
- **Mask:** `1d100e105e542778`
- **Key:** `d3x://vault`

**Logic:**
For each byte at index `i`, XOR the mask byte with the corresponding key byte and a dynamic value:
`Result[i] = Mask[i] ^ Key[i % len(Key)] ^ ((i * 5 + 11) & 0xff)`

**Result:** `r3********7`

---

## 4. Final Flag Construction
Concatenating the three segments:
`m0******` + `rt********` + `r3********7`

**Full Flag:**
`ZeroX{m0************************_x7}}`
